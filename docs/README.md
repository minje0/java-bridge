## 기능 목록
- [ ] 다리 길이를 입력 받는다.
    - [ ] "다리의 길이를 입력해주세요." 출력
    - [ ] 다리 길이는 3~20 사이의 숫자만 입력 가능
- [ ] 입력 받은 길이의 다리를 생성한다.
- [ ] 이동할 칸을 입력 받는다.
    - [ ] "이동할 칸을 선택해주세요. (위: U, 아래: D)" 출력 (OutputView)
    - [ ] U or D 만 입력 가능 (InputView)
- [ ] 이동한 모습 출력 (OutputView)
- [ ] 게임의 재시작 여부 입력 받는다.
    - [ ] "게임을 다시 시도할지 여부를 입력해주세요. (재시도: R, 종료: Q)" 출력 (OutputView)
    - [ ] R or Q 만 입력 가능 (InputView)
- [ ] 최종 게임 결과를 출력한다.
- [ ] 게임 성공 여부를 출력한다.
- [ ] 총 시도한 횟수를 출력한다.
